Project Overview & Objectives

Comparative Genomics at Scale

Analyzing whole-genome alignments generated by the TOGA (Tool for Genome Alignment) pipeline to discover loci under selection across mammalia.

Screening Selection Events

Using site-level (MEME) and branch-level (aBSREL) statistics to pinpoint transient and lineages-specific adaptive shifts.

Three Evolutionary Aims:

  • Aim 1: Population Size & Purifying Selection
    Test if clades with large effective population sizes ($N_e$) show stronger constraint (lower $dN/dS$).
  • Aim 2: Dynamic Selection Hotspots
    Verify whether positive selection targeting is conserved or shifts across clades.
  • Aim 3: Taxon Sampling Return Curve
    Analyze how species density in alignments affects statistical power.

Computational Strategy

To screen selection events genome-wide, we compiled codon statistics in SQLite. Selection calls were defined as non-synonymous changes occurring on terminal leaf branches with a site-level positive selection p-value $P < 0.05$ (using HyPhy MEME). Benjamini-Hochberg False Discovery Rate (FDR) correction controls error rates at $Q \le 0.05$.

Echolocation Selection Screen

We set up an evolutionary association screen to detect genes where positive selection is enriched on terminal branches of echolocating lineages versus control lineages.

Target Lineages ($N = 110$ species)

Bats: Yangochiroptera & Yinpterochiroptera (excluding megabats).
Toothed Whales: 26 odontocete genera (e.g. Tursiops, Phocoena, Physeter).

Control Lineages ($N = 28$ species)

Megabats: Non-echolocating Pteropodidae (e.g., Pteropus, Rousettus).
Baleen Whales: 6 non-echolocating mysticete genera (e.g., Balaenoptera, Megaptera).

Methodological Details:

  • Synonymous Control: Filtered out synonymous mutations (only non-synonymous selection tracked).
  • Taxonomy Mapping: Integrated genus-based checks to prevent taxonomic misclassifications.
  • Branch Control: Restricted selection calls to leaf branches (representing active, extant phenotypes).
  • Statistical Test: Fisher's Exact Test per gene, with Benjamini-Hochberg FDR correction.

Hypothesis 1: Population Size & Purifying Selection

Population Genetics Theory

Lineages with large effective population sizes ($N_e$) should undergo more efficient natural selection, leading to stronger purifying constraint (lower background $dN/dS$, or ω).

The Short-Branch Mirage (Bias)

Species-dense clades (e.g. Primates, Cetacea) have systematically shorter terminal branches. On short branches, synonymous substitutions ($dS$) are frequently zero, which mathematically inflates MLE ω estimates (division by near-zero).

Divergence & Bias:

  • Primates/Cetacea branch lengths: Median $\approx 0.0009$ substitutions/site.
  • Rodentia/Chiroptera branch lengths: Median $\approx 0.006$ substitutions/site.
  • Unfiltered background analysis shows an artificial negative correlation ($r \approx -0.58$) between terminal branch length and estimated background omega.
Slide 10 of 18 Purifying Selection Bias

Hypothesis 1: Controlling for Branch Length Bias

To resolve this math artifact, we implemented a branch-length filtering threshold, keeping only terminal branches with length > 0.003 substitutions/site.

Effect of Filter:

Applying the filter successfully removes the mathematical inflation of $\omega$ on short branches, exposing true biological purifying constraint differences across mammalian clades.

Hypothesis 2: Dynamic Selection Clade-Shifts

Do evolutionary adaptive landscapes shift between deep mammalian lineages (epistasis), or remain constant across structural hotspots?

Stratified Permutation Test (49,013 sites):
Shuffled positive selection branches across tree topologies (10,000 permutations) to create a controlled null expectation.
  • Shared Selection Sites: Observed 91.84% is significantly lower than the expected 94.36% (Ratio = 0.973, $P = 9.29 \times 10^{-201}$).
  • Euarchontoglires-only Sites: Enriched by ~2-fold (Observed 4.05% vs Expected 2.14%).
  • Laurasiatheria-only Sites: Enriched by 16% (Observed 3.97% vs Expected 3.42%).

Shared Selected Sites (Observed vs Expected)

The overlap is significantly smaller than the null expectation, proving that selection targets shift due to epistasis and niche differentiation.

Hypothesis 3: Species Return Curves

Does adding more genomes to an alignment indefinitely yield linear returns in selection detection power?

The Biphasic Return Curve:
Selection density peaks at intermediate species counts (500–600 genomes, density = 0.024) but drops by 4-fold in genes represented across 700+ genomes (density = 0.005).
  • Immutable Housekeeping Genes: Genes annotated across all 700+ species represent core housekeeping machinery (e.g. ribosomes, translation, transcription).
  • These genes are under intense purifying selection and rarely tolerate non-synonymous changes, which explains the drop-off in positive selection density.
Slide 14 of 18 Biphasic Sampling Dynamics

Hypothesis 3: Tree Length Drives Power

If species count is a poor proxy for selection power, what drives selection discovery?

Mutational Depth (Tree Length):
Tree Length represents the total evolutionary divergence in the tree (substitutions/site). It correlates strongly with selection density ($r_s = 0.30$, $P = 3.05 \times 10^{-212}$).
  • Logarithmic models fit the curve significantly better than linear models ($R^2 = 0.0069$, $AIC = -62640.09$), showing a gradual return saturation.
  • Takeaway: Alignments need evolutionary depth (tree branch diversity) rather than sheer species counts to maximize adaptive site discovery.

Scaling Selection: PhyloAxialTransformer

Standard comparative genomic likelihood optimizations (e.g. MEME, aBSREL) require hours per gene and fail to scale to VGP limits (thousands of species).

Machine Learning-Assisted Selection Screens

We developed a regression transformer (PhyloAxialTransformer) that bypasses expensive numerical optimizations to predict selection statistics.

  • Inputs: Combines sequence token embeddings, branch patristic distances, and multidimensional scaling (MDS) tree coordinates.
  • Performance: Predicts regularized LRT statistics, filtering alignment/sequencing noise and accelerating analysis speeds by orders of magnitude.
